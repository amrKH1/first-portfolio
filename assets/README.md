## Image Guidelines

### For Profile Photos:
- Use high-quality, professional photos
- Ensure good lighting and clear background
- Consider using a consistent style/filter across photos
- Optimize for web (compress without losing quality)

### For Project Screenshots:
- Use actual screenshots of your projects when possible
- If projects don't exist yet, create mockups or use placeholder images
- Ensure screenshots show the key features of each project
- Maintain consistent aspect ratios for better visual harmony

### Optimization Tips:
- Compress images to reduce file size while maintaining quality
- Use WebP format for better compression (with fallbacks)
- Consider using responsive images for different screen sizes
- Add alt text for accessibility
